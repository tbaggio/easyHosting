# easyHosting

**OpenHostingAPI** can :
- Manage **VMs** with OpenNebula (CREATE UPDATE DELETE READ)
- Deploy **services** on VMs with Ansible
- Manage **Users** (CREATE UPDATE DELETE READ)
- Secure **authentification** (JWT)
- Manage **Group & Roles** for Users

### API Requirements
- Open Nebula 5
- Ansible
- NodeJS LTS
- Angular 9
